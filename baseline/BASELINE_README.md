# Preserved construct-validity baseline

This directory is a byte-preserved copy of the original generated corpus, derived outputs and figures. It is retained so that the original seeded baseline is not lost while the independent-validation design evolves.

The legacy data group `mixed` means **rule-perturbed synthetic**. Its programmatic transformations were not performed by humans. Historical result keys using `mixed` or `machineorigin` are retained here only for backward compatibility and must not be used as manuscript-facing terminology.

The baseline is a controlled construct-validity demonstration, not external validation or an estimate of operational performance.
